# ⚔️ HOBBYTAN-COUNCIL: Build in Public (HobbytanAI Core OS)

> **"당신과 AI기술 사이, 하비탄AI."**

본 레포지토리는 하비탄AI의 **지능적 중추(Core OS)**이자, 그 구축 과정을 실시간으로 제품화하는 **BIP(Build in Public) 서비스의 실체**입니다.

## 🏛️ TAN: Thinking Assembly Network
**TAN**은 "Thinking Assembly Network"의 약자로, 인간의 형상을 하고 있지만 집단 지성 네트워크에 연결된 12인의 정예 에이전트 그룹을 의미합니다.
이들은 실리콘밸리의 기술 스타트업 전문가 페르소나를 가지며, 오른쪽 눈 아래에 **[TAN]**이라는 문신이 새겨져 있습니다.
